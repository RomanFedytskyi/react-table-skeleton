# Security

If you discover a security vulnerability in this project, please report it privately to the maintainers so it can be addressed before public disclosure.

Preferred reporting:
- Open a private issue in this repository and mark it `security` (or mention maintainers by name).
- If you'd rather email, send details to the project owner email listed on the GitHub profile.

Please include steps to reproduce, impact, and any suggested mitigations. Do not publish exploit details until a fix is released.
