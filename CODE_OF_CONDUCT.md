# Code of Conduct

This project follows a standard Code of Conduct to foster an open and welcoming community. Please be respectful, considerate, and professional. Unacceptable behaviour (harassment, hate speech, threats, etc.) will not be tolerated.

If you experience or witness unacceptable behaviour, please contact the maintainers via the repository issue tracker or email the project owner.

Short summary:
- Be kind and professional.
- Assume good intent.
- Report issues to maintainers.
